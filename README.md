# Mini E-Commerce Cart System

## Project Overview

The Mini E-Commerce Cart System is a responsive web application developed using React.js. The project was designed to provide a simple and efficient online shopping experience by allowing users to browse products, add items to a shopping cart, manage product quantities, and calculate the total cost of selected items.

This application serves as a prototype for a full-scale e-commerce platform and demonstrates the implementation of essential shopping cart functionalities using modern frontend development techniques.

## Features

* Product listing using API integration
* Add products to cart
* Remove products from cart
* Increase and decrease product quantities
* Automatic total price calculation
* Responsive and user-friendly interface
* Real-time cart updates
* Professional UI design using CSS

## Technologies Used

* React.js
* JavaScript (ES6+)
* HTML5
* CSS3
* Fake Store API
* React Hooks (useState, useEffect)

## Project Objectives

The primary objective of this project is to create a lightweight e-commerce application that demonstrates core shopping cart operations. It helps users easily manage products and provides a foundation for future enhancements such as authentication, payment integration, order tracking, and database connectivity.

## Functional Modules

### Product Management

* Fetches product data from an external API.
* Displays product images, names, and prices.

### Cart Management

* Add products to the cart.
* Remove products from the cart.
* Update product quantities.

### Price Calculation

* Calculates the total cost based on selected products and quantities.
* Updates totals dynamically whenever cart data changes.
